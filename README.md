# Dólar BCV Monitor 💎

Monitor en tiempo real del dólar BCV (oficial), paralelo y USDT en Venezuela.

## Funcionalidades
- 💵 Tasa USD BCV oficial (hoy y ayer)
- 📊 Variación porcentual con gráficas
- 💱 Dólar Paralelo y USDT
- 🔄 Calculadora de divisas (USD, EUR, USDT, VES)
- 📈 Historial del dólar BCV desde 2023
- 💀 Calculadora de pérdida en poder adquisitivo

## APIs utilizadas
- [DolarVZLA](https://dolarvzla.com) - Tasa BCV oficial
- [DolarAPI](https://ve.dolarapi.com) - Dólar paralelo e historial

## Tecnologías
- HTML5, CSS3, JavaScript vanilla
- Chart.js para gráficas
- Sin dependencias backend
